LangGraph + Multi-Agent RAG + Education AI

🎓 Multi-Agent Education RAG

A production-style conversational RAG system built using LangGraph, FastAPI, FAISS, BM25, Streamlit, and OpenAI. The system supports conversational memory, query rewriting, hybrid retrieval, grounded generation, verification, and evaluation.

View GitHub Repository Live Demo View Architecture

✨ Key Features

This project demonstrates a full conversational multi-agent RAG pipeline with grounding, evaluation, and observability.

🧠 Query Rewriting

Converts follow-up conversational questions into standalone retrieval-ready queries.

🔀 Multi-Agent Workflow

Uses LangGraph agents for routing, retrieval, generation, quiz creation, and verification.

🔍 Hybrid Retrieval

Combines FAISS semantic search and BM25 keyword retrieval for better accuracy.

✅ Grounded Generation

Generates answers strictly from retrieved educational content to reduce hallucinations.

📊 Confidence Scoring

Exposes retrieval similarity scores and qualitative confidence labels.

🧪 Evaluation Pipeline

Includes verifier-agent evaluation inspired by RAGAS, DeepEval, and LLM-as-a-judge approaches.
